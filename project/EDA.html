專案經歷

WEB - 專櫃直售平台

由於義大購物中心原僅有線下銷售，非線下部分則透過專櫃積極與顧客維繫關係以利回購。為提高購買意願及增進收入，特別建置直售平台。平台可新增訂單，並串接台銀虛擬ATM及聯合信用卡金流功能，讓顧客可透過網路購買商品，提高整體營運效率，並提升顧客體驗。

技術使用

Vue.js
C#
Bootstrap4
獨立開發

WEB - 遊樂園快速購票平台

因每逢假日遊樂園人潮眾多，為減少現場排隊等候時間，建置快速購票平台，讓顧客可透過網路購買門票，並可於現場直接掃描QR Code進入遊樂園。

技術使用

Vue.js
C#
Bootstrap4

APP - 義大Easy購 2.0 APP

為打造行動會員卡進而開發跨平台 APP ，提供會員更多購物便利，在上面可接收最新及專屬優惠。

技術使用

Cordova
C# API

WEB - 飯店住客樂園門票派送系統

為提升飯店住客體驗及減少飯店第一線人員工作內容，建置住客樂園門票派送系統，讓住客可透過網路預約樂園門票，並透過簡訊或是 Email 派送入園憑證，於現場直接掃描QR Code進入遊樂園。

技術使用

C#
Bootstrap5
獨立開發
