Write Number in Expanded Form

題目情境​

將題共給予的數字已每個位數去做擴展，這題看題目敘述有點難理解，直接看範例就清楚多了

expandedForm(12); // Should return '10 + 2'
expandedForm(42); // Should return '40 + 2'
expandedForm(70304); // Should return '70000 + 300 + 4'

如果遇到 0 就略過不理它。

解題思路​

這題要考的就是對於位數 0 的處理，像5位數就是5位數的數字 * 10000 (4個0)
0的數字會比位數少一碼，有這個邏輯就可以來寫 code 了

首先一樣用將題目傳入的數字轉換成陣列，number無法直接轉換成陣列，所以要先轉換成字串 toString() 再來使用 .map() 來產生新陣列，如果位數為 0 就回傳空值不理它，若不是就加上剛剛說的 0 的處理，因為是字串所以使用 .repeat()
傳入的參數就是 array.length - 1 - index，要適時扣除 index 的數字才會加上正確的 0 ，不然只會加上陣列數 - 1
5位數就是 5 - 1 - 0 只會加上 4 個 0，最後再用 .filter() 拿去除 map 出來為空值的內容，再用 .join() 組字串。

function expandedForm(num) {
  return num.toString().split('')
        .map((char,index,array) => char == "0" ? "" : char + "0".repeat(array.length - 1 - index))
        .filter(item => item)
        .join(" + ")
}

知識點​

  • Array.prototype.map()
  • Array.prototype.filter()
  • Array.prototype.join()
