Detect Pangram

題目情境​

所謂 Pangram 就是全字母句，在句中 a~z 的英文字母都會出現並不分大小寫
寫一個 function 傳入字串參數(句子)，並判斷是否為 Pangram，忽略數字及標點符號。

解題思路​

直覺連結就是使用陣列最為方便，先新增一個 a-z 的陣列來做遍歷 'abcdefghijklmnopqrstuvwxyz'.split(''); 再使用 indexOf() 來檢查是否每個單字都出現過一次，不曾出現會回傳 -1 ，所以只要大於 -1即可

function isPangram(string){
  let chars =  'abcdefghijklmnopqrstuvwxyz'.split('');
  return chars.map(char => string.toLowerCase().split('').filter(_char =>
     _char).indexOf(char) > -1).filter(isTrue => isTrue).length == 26 
}

這邊曾經踩了一個糊塗坑，就是只判斷小寫導致測試時失敗，所以要記得把傳入的參數先轉為小寫 toLowerCase()
成功後就開始想想是否有更簡潔的寫法，雖然可以達成目的但寫起來稍為的醜，翻了一下 mdn 發現了 every() 更能達成目的

function isPangram(string){
  let chars =  "abcdefghijklmnopqrstuvwxyz".split("")
  return chars.every(char=> string.toLowerCase().indexOf(char) !== -1 );
}

知識點​

  • String.prototype.split()
  • Array.prototype.map()
  • Array.prototype.every()
  • String.prototype.toLowerCase()
