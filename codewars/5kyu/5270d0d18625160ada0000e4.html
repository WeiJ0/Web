Greed is Good

題目情境​

Greed 是一個擲骰子的遊戲，會擲出 5 個骰子，點數會依照特定規則來計分，規則如下：

點數積分
3 個 11000
3 個 6600
3 個 5500
3 個 4400
3 個 3300
3 個 2200
1 個 1100
1 個 550

若有 bonus 加分後要把 bonus 的骰子扣除再去計算分數 範例：

5 1 3 4 1 = 250
2個1 (100 x 2) + 1個5 (50 x 1)
1 1 1 3 1 = 1100
3個1 (1000) + 剩下的1個1 (100 x 1)

解題思路​

我這邊先把 bonus 的部份定義出來

let bonus = {
  1: 1000,
  2: 200,
  3: 300,
  4: 400,
  5: 500,
  6: 600,
};

再把骰出來的結果依照點數大小存在另外一個陣列中 numCount

let numCount = []; // [1的個數, 2的個數, 3的個數, 4的個數, 5的個數, 6的個數]

for (let i = 0; i < 6; i++) {
  numCount[i] = dice.filter((num) => num == i + 1).length; // + 1 是因為 array 的 index 和目標點數剛好差 1
}

再來再用迴圈去判斷是否有 bonus 並加分，若有要扣掉 bonus 的骰子扣除再去計算分數

for (let num in bonus) {
  if (numCount[num - 1] >= 3) {
    //點數有三個的話做 bonus
    points += bonus[num];
    numCount[num - 1] -= 3; // 扣除三個骰子
  }
}

points += numCount[0] * 100; // 剩下的 1
points += numCount[4] * 50; // 剩下的 5

完整程式​

function score(dice) {
  let numCount = [];
  let points = 0;
  let bonus = {
    1: 1000,
    2: 200,
    3: 300,
    4: 400,
    5: 500,
    6: 600,
  };

  for (let i = 0; i < 6; i++) {
    numCount[i] = dice.filter((num) => num == i + 1).length;
  }

  for (let num in bonus) {
    if (numCount[num - 1] >= 3) {
      points += bonus[num];
      numCount[num - 1] -= 3;
    }
  }

  points += numCount[0] * 100;
  points += numCount[4] * 50;

  return points;
}

知識點​

  • Array.prototype.filter()
  • for...in
