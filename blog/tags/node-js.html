使用 puppeteer 取得 ig 圖片 & 內容

September 10, 2022 · 5 min read
WeiJie

前言​

在年初因應公司官網改版，需要在官網中嵌入 instagram 的資料，於是上網找了些資料發現 ig 有提供公開的 JSON API。

https://www.instagram.com/公開帳號的id/?__a=1

整體流程為使用 puppeteer 到 ig 抓取最新的九則貼文圖片及內容，因為 JSON API 中的圖片連結無法在限定的網域被讀取，所以圖片得另外處理。

不過近期被告知貼文一直沒有更新，重新 run puppeteer 才知道 API 掛了。原因看來是 meta 收購 ig 後有做 API 調整，目前需要以該帳號的權限到 Facebook 申請 Token，不過公司的架構權限的問題還要另外提出說明及申請，所以就把所有內容都用 puppeteer 來抓取。

需要使用的套件​

  • fs
  • request
  • puppeteer puppeteer 說明文件

功能需求​

  1. 下載最新貼文的 9 則圖片
  2. 取得近九則的貼文內容並轉為 JSON，並與 9 張圖片連結比對
