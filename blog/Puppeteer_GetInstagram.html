使用 puppeteer 取得 ig 圖片 & 內容

September 10, 2022 · 5 min read
WeiJie

前言​

在年初因應公司官網改版，需要在官網中嵌入 instagram 的資料，於是上網找了些資料發現 ig 有提供公開的 JSON API。

https://www.instagram.com/公開帳號的id/?__a=1

整體流程為使用 puppeteer 到 ig 抓取最新的九則貼文圖片及內容，因為 JSON API 中的圖片連結無法在限定的網域被讀取，所以圖片得另外處理。

不過近期被告知貼文一直沒有更新，重新 run puppeteer 才知道 API 掛了。原因看來是 meta 收購 ig 後有做 API 調整，目前需要以該帳號的權限到 Facebook 申請 Token，不過公司的架構權限的問題還要另外提出說明及申請，所以就把所有內容都用 puppeteer 來抓取。

需要使用的套件​

  • fs
  • request
  • puppeteer puppeteer 說明文件

功能需求​

  1. 下載最新貼文的 9 則圖片
  2. 取得近九則的貼文內容並轉為 JSON，並與 9 張圖片連結比對

puppeteer 下載圖片​

ig.js
// url 圖片連結 , filename 圖片名稱(路徑)
const request = require("request");

const download = (url, filename) =>
  new Promise((resolve, reject) =>
    request.head(url, (err, res, body) => {
      request(url).pipe(fs.createWriteStream(filename)).on("close", resolve);
    })
  );

範例取得小王子專業文章圖片​

  1. 前往 IG 帳號頁面
const browser = await puppeteer.launch();
const page = await browser.newPage();
// 電腦版及行動裝置的 html 架構可能不同，這邊以電腦版為例
await page.setViewport({
  width: 1920,
  height: 1020,
  isMobile: false,
});
await page.setUserAgent(
  "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/97.0.4692.71 Safari/537.36 Edg/97.0.1072.55"
);

// 登入功能 Start
// 有時會因帳號設定需要登入才能到帳號頁面，若需要則自行執行下方登入功能。
await page.goto("https://www.instagram.com/accounts/login/");
await page.waitForSelector('input[name="username"]');
await page.click('input[name="username"]');
await page.keyboard.type("IG帳號");
await page.click('input[name="password"]');
await page.keyboard.type("IG密碼");
await page.click('button[type="submit"]');
await page.waitForNavigation();
// 登入功能 End

await page.goto("https://www.instagram.com/lepetitprinceofficiel/");
await page.waitForSelector("article"); // 等待 article selector 產生
  1. 使用 javascript console.log 來取得相關資料
// 頁面中 console.log 的資料會存入 result
let result = "";
page.on("console", (msg) => {
  result += msg.text();
});

// 於頁面執行 js
await page.evaluate(async () => {
  // 可參閱 ig 頁面 html 結構，若未來調整則調整 selector
  let arr = Array.from(document.querySelectorAll("article a"));
  let output = [];
  arr.forEach((item, index) => {
    if (index < 9)
      // 只取 9 則
      output.push({
        index: index + 1,
        img: item.querySelector("img").src, //圖片連結
        content: item.querySelector("img").alt, //圖片貼文內容，IG 會放在圖片 alt
        link: item.href, //貼文連結
      });
  });
  console.log(JSON.stringify(output)); // 將 output 轉成字串存入 result
});
  1. 圖片下載 & JSON 檔轉出
// console.log 專出資料在轉回 JSON
let imgArr = JSON.parse(result);

// 迴圈下載圖片
imgArr.forEach((item, index) => {
  download(item.img, `${index + 1}.jpg`);
});

// 最後將資料匯出目錄下 ig.json
fs.writeFileSync("ig.json", JSON.stringify(result), (err) => {
  if (err) {
    console.error(err);
    return;
  }
});
