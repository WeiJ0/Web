利用 Google Apps Script 監控網站，並使用 Line Notify 通知

August 21, 2022 · 5 min read
WeiJie

前言​

前幾天公司發生外部 DNS 服務異常導致官網、APP 這些對外服務通通都出現問題，這才想到目前對於這些系統的監控都是來自內部網路
而像 DNS 這種外部網路才能發現的事情倒是漏掉了，於是想了想，還是免費的 Google Scripts 最好用。

建立新專案​

Apps Script 網站入口

在右上方按下新專案，畫面中會預設一個 function myFunction()，這就是要執行程式的地方

Google Apps Script 的寫法與 javascript 差不多，只是又多了一些 Google 幫你寫好的函式可以使用
所以這邊我們就用 Fetch 來 GET 網站看看是不是能正確回傳 200 的 Response Code

Google Apps Script
// 若有多網站則新增至 webs 物件即可
const webs = [{ Name: "example", URL: "http://example.com/" }];

let errorWebs = [];

function webCheck() {
  webs.forEach((item) => {
    try {
      let response = UrlFetchApp.fetch(item.url);
      if (response.getResponseCode() != 200) errorWebs.push(item.Name); // 若是 Response Code 不是 200 就加入錯誤陣列
    } catch (err) {
      // 這邊會使用 try catch 防止 fetch 出現異常錯誤
      errorWebs.push(item.Name); // 錯誤也加入提醒讓我們檢查
    }
  });

  if (errorWebs.length > 0) {
    let errString = "目前異常：" + errorWebs.join(",");
    // 目前異常： example
    // 這邊在做通知處理，把上述這段錯誤訊息寄出
  }
}

錯誤通知​

錯誤處理使用 Line Notify 來發送通知，首先先到 Line Notify 後台來建立憑證
登入後從右上角的名稱功能列進去 個人頁面，再點擊下方有一個發行權杖的按鈕

這邊需要輸入選擇兩個欄位

  • 權杖名稱：他在 Line Notify 推播給你的時候會顯示，讓你能分辨是什麼服務使用
  • 接收通知的聊天室：若你是要自己收到就選擇最上方的 「透過 1 對 1 聊天接收 Line Notify 的通知」
    tip

    也可以在其他人的聊天室或群組中接收，但就是要在群組中加入 Line Notify 這個官方帳號

送出後就可以取得 Token，這個 Token 是不會過期的可以一直使用

這時候再回到 Google Apps Script 這邊，用 Fetch Post 資料給 Line Notify API 進行通知 Token 的部分是使用 Bearer Token，詳細文件可以參照 Line 官方文件

function lineNotfiy(message) {
  UrlFetchApp.fetch("https://notify-api.line.me/api/notify", {
    headers: {
      Authorization: "Bearer " + [lineToken],
    },
    method: "post",
    payload: {
      message: message,
    },
  });
}

function 寫完後就是把錯誤訊息給傳進來就可以試著發送了，執行的話則按編輯器上方有一個「執行」，
右邊的 Function 要記得改成 WebCheck

if (errorWebs.length > 0) {
  let errString = "目前異常：" + errorWebs.join(",");
  lineNotfiy(message);
}

不過執行前，因為這個程式是我們自己寫的，並非來自於 Google 本身，所以會提醒並沒有經過官方認證的警告
在跳出的視窗選擇「進階」 以及 前往「你的專案」(不安全)

允許後就會執行程式了，而如果有錯誤就會收到 Line Notify

確認完程式沒問題後就要來設定排程讓程式自動執行，在 Google 編輯器左邊有一個觸發條件，點下後再選擇新增觸發條件
條件的部分可以設定使用 時間驅動 可以選擇每月、每週、每日、每小時最短到每分鐘的計時器可以選擇，完成後按下儲存就大功告成了

Google Apps Script 程式碼​

const LINE_TOKEN = [TOKEN];
const webs = [{ Name: "example", URL: "http://example.com/" }];

let errorWebs = [];

function webCheck() {
  webs.forEach((item) => {
    try {
      let response = UrlFetchApp.fetch(item.url);
      if (response.getResponseCode() == 200) errorWebs.push(item.Name);
    } catch (err) {
      errorWebs.push(item.Name);
    }
  });

  if (errorWebs.length > 0) {
    let errString = "目前異常：" + errorWebs.join(",");
    lineNotify(errString);
  }
}

function lineNotify(message) {
  UrlFetchApp.fetch("https://notify-api.line.me/api/notify", {
    headers: {
      Authorization: "Bearer " + LINE_TOKEN,
    },
    method: "post",
    payload: {
      message: message,
    },
  });
}
