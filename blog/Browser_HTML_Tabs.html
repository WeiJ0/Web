關於瀏覽器頁籤的那些小事，看看使用者是否離開你的頁籤

September 14, 2022 · 5 min read
WeiJie

前言​

在 Facebook 的頁面上，若是有人傳送 Message 給你，剛好頁籤在 Facebook 上的話會直接由右下方談出 Message 聊天室，而在其他分頁時則會看到 Facebook 分頁出現了未讀數字以及某某某傳送訊息給您的字樣。最近剛好有這樣的需求，上網蒐集了一些做法並且紀錄下來。

由於未使用 websocket 的技術，所以就使用頁籤的切換事件當作模擬來動態修改。

介紹頁籤​

左邊會有一個小的圖示為 favicon.ico，為 Favorites Icon 的縮寫
早期運用在加入我的最愛資料夾中的縮圖，後期至今已經成為瀏覽器頁籤上的縮圖，而像是 Google 搜尋或是分頁也會顯示該圖片，檔案的尺寸常見為 16x16 32x32，若設定需要將原圖片轉檔為 .ico 檔，通常置放於網站的跟目錄下，如於其他位置須於 <head> 中指定

<link rel="icon" href="/Web/img/favicon.ico" />

而右邊文字的部分則為網站的標題，通常也會一並建置於 <head> 中，也可以使用 js 動態進行調整

<title>Blog | WJ 開發踩雷&新鮮東西筆記站</title>

動態切換標題​

那除了原本就寫好放在 head 裡的 title，若今天是動態的網頁像是部落格，會將文章標題後方再加上部落格名稱，除了一篇一篇修改外，用 js 的方式調整更加的快速

document.title = "換這個標題";

沒錯，就這樣短短的一行，像是 Facebook 舊式頁面上的 websocket 接收到有人方送 Message 就會執行 title 的切換，而已讀訊息再切換回 Facebook。 這邊就用切換頁籤事件當作範例吧

window.addEventListener("focus", () => {
  document.title = "你現在正在看著呢";
});
window.addEventListener("blur", () => {
  document.title = "你怎麼不看了呢";
});

這邊綁定 window 的 focus 及 blur 事件，分別觸發在我進入以及離開頁籤的時機點。

相同的需求使用 visibilitychange 一樣能做到

document.addEventListener("visibilitychange", () => {
  let status = document.visibilityState;
  document.title = status == "visible" ? "你現在正在看著呢" : "你怎麼不看了呢";
});

動態切換圖示​

比較陽春的就是先用 js 尋找是 icon 的 link 項目，並動態替換掉圖片網址

var link =
  document.querySelector("link[rel*='icon']") || document.createElement("link");
link.type = "image/x-icon";
link.rel = "shortcut icon";
link.href = "https://docusaurus.io/zh-CN/img/docusaurus.png";
document.getElementsByTagName("head")[0].appendChild(link);

把上面程式碼用 F12 主控台開起來貼上，這個頁面的圖示應該就變成小恐龍了

如果要在稍微進階的功能可以使用下面這個 plugin

favico.js

favico.js 提供了圖示的更換、計數功能的背景文字顏色，甚至還設置動畫。

將 icon 設定為某個 element 圖片​

var favicon = new Favico();
var image = document.getElementById("imageId");
favicon.image(image);

將 icon 設定為 html5 的 video​

var favicon = new Favico();
var video = document.getElementById("videoId");
favicon.video(video);
// favicon.video('stop');

將 icon 設定為網路攝影機的內容​

var favicon = new Favico();
favicon.webcam();
//favicon.webcam('stop');

設置計數 Badge​

var favicon = new Favico({
  animation: "slide",
});
favicon.badge(1); // 數量

詳細的 setting object 如下

屬性說明
bgColor背景色 (#d00)
textColor文字色 (#fff)
fontFamily字體 (Arial, Verdana, Times New Roman, serif, sans-serif)
fontStyle字體粗體、斜體 (normal, italic, oblique, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900)
type為圓形或方形 (circle, rectangle)
position顯示位置 (up, down, left, upleft)
animation變更時動畫效果，可於favico.js 網站中預覽 (slide, fade, pop, popFade, none )
elementIdelementId 如果需要替換成 element 中的圖片使用
