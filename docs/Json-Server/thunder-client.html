json-server in vscode thunder-client

前言​

首先，先來介紹 thunder-client 是什麼，他與 postman 相同都是一款 API 測試工具，
他的優點是可以直接在 vscode 中使用，不需要另外切換視窗，並且可以直接在 vscode 中編輯 API 測試，不用另外開啟編輯器。

gif 來自於 marketplace - Thunder Client

將 json-server 與 thunder-client 連結​

thunder-client 和 postman 一樣，都可以針對多個 API 組成一個集合，方便管理及協作時直接匯出匯入
而這邊我使用了 js 寫了一個將 json-server 的 db.json 轉成 thunder-client 集合可以使用的格式。

thunder-client-json-server - github page

並且將每個資料集分別產出 GET POST PUT PATCH DELETE 的 API 測試，方便使用者直接測試。
在 POST PUT PATCH 需傳入 body 時，會自動產生一個 body 的範例，源至於你的 db.json第一筆資料。

後續針對 Thunder Client 提供的相關功能也會再找時間單獨寫一篇內容，介紹他完整的功能。
