將Oracle中Long Raw用ashx輸出

前言​

這幾天剛好有一個需求是要存取會員的照片，以往都是在後台伺服器的某個資料夾，只要掛載虛擬目錄就能完成
不過這次系統是將照片用 Long Raw 存入 db，第一次遇到覺得新鮮，也順便紀錄 C# 如何將二進制的資料再轉回圖片

建立一支 ashx 檔​

ashx 檔就是沒有畫面的 aspx 檔，所以可以用來自由輸出各種形式的資料，像是 JSON 和這次要實作的圖片檔都可以
首先會先定義這次要輸出的資料 ContentType 為何

context.Response.ClearContent();
context.Response.ContentType = "image/Jpeg";

接下來就是從資料庫取出 Long Raw 的內容就不寫出來，暫且一個用一個變數 PhotoImage 儲存
將二進制資料轉型成 Byte[]

Object PhotoImage = '二進制資料...'
Byte[] img = (Byte[])PhotoImage;

再來用 MemoryStream 及 Bitmap 來繪製圖片

MemoryStream ms = new MemoryStream(img);
Bitmap bitmap = new Bitmap(ms);
bitmap.Save(ms, System.Drawing.Imaging.ImageFormat.Jpeg);

最後再將 MemoryStream 用 BinaryWrite 的方式輸出就可以了

context.Response.BinaryWrite(ms.ToArray());
bitmap.Dispose();
context.Response.End();

完整程式碼​

public void ProcessRequest(HttpContext context)
{
    Object PhotoImage = '資料庫處理取出二進制資料'
    Byte[] img = (Byte[])PhotoImage;

    MemoryStream ms = new MemoryStream(img);
    Bitmap bitmap = new Bitmap(ms);
    bitmap.Save(ms, System.Drawing.Imaging.ImageFormat.Jpeg);
    context.Response.ClearContent();
    context.Response.ContentType = "image/Jpeg";
    context.Response.BinaryWrite(ms.ToArray());
    bitmap.Dispose();
    context.Response.End();
}
