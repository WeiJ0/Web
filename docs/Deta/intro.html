Deta，Heroku 替代方案之一

前言​

Heroku 在今年 8 月底的時候宣布，會逐漸淘汰各種免費的服務，像是最多人使用的 Dynos 及 Postgres
頓時程式圈哀鴻遍野，少了這麼一個棒的免費平台實在可惜，所以找替代方案也成為這陣子大家的工作之一
這個系列要介紹的 Deta 正是部分可以替代的方案之一，不過他只支援 node 及 python 兩種語言使用
並且號稱 永遠免費 ，雖然這個永遠不知道是不是真的永遠，不過在這段期間還是能讓大家練練程式，建立一個微型的網站服務

服務 - Deta Micros​

Deta Micros 是一個雲端運算的微型伺服器，可以上傳 Node Express 進行運作，只需要專注於 Express 的開發
其他相關的事情就交給 Deta 處理即可，也提供了簡易的後台 UI 做所有設定。

既然是微型服務，自然是有所限制。

  1. 程式碼及相關檔案上傳大小的上限為 250 MB，包含 NPM 的項目。
  2. 執行的記憶體只有 512 MB，若需要增加需要 Mail 官方說明原因及目的。
  3. TimeOut 時間為 10 秒，若需要增加需要 Mail 官方說明原因及目的。
  4. 不支援 Websokets 及 long-running 程式，所以也無法使用 socket.io 或是相關的機器人服務 Discord bots
  5. python 的部份 packages 無法使用 (Google & Firebase)
  6. 支援 python 3.7, 3.8, 3.9 & Nodejs 12.x 14.x

微型資料庫 NoSQL - Deta Base​

Deta Base 是一個快速且安全的 NoSQL 資料庫，託管於 AWS 上且資料會再進行一次加密
並且提供了一個 UI 於後台界面上，更容易了解資料庫的所有紀錄。

而除了使用 Deta Micros 架設微型伺服器外，Deta 也另外提供了 HTTP API，即使不懂得 Node Express 還是可以使用。

微型雲端硬碟 Deta Drive​

Deta Drive 提供了每個用戶可上傳文件的服務，而限制為 10GB，也提供後台 UI 及 HTTP API 讓使用者更方便使用。

結論​

雖然 Deta 有著種種的限制，不過出自於免費提供，這點限制也是無法避免，服務內容雖然無法與Heroku相比，但拿來練手程式，
應該還是綽綽有餘。
